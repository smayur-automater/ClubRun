# ClubRuns — Feature Backlog: Tracking & Monetization

Owner: Marketing + CEO agents. Prioritized advisory (2026-07-02) answering: what else should runners be able to track, and what does the business earn from each. Sequenced against `docs/ROADMAP.md` milestones; nothing here ships before its dependency does.

## Tier 1 — Tracking depth that converts to ClubRuns+ (build at M3, gate at M5)

| Feature | Runner value | Business value |
|---|---|---|
| **Pace zones & effort analysis** (easy/tempo/threshold splits per run) | Know if you're training or just tiring | The single most proven fitness-app upsell (Strava/Garmin precedent) — headline ClubRuns+ feature |
| **PR progression curves** (5K/10K/half trend lines) | See yourself getting faster | Premium chart; free users see current PRs, plus sees the journey |
| **Heart-rate + cadence** (wearable BLE/HealthKit/Health Connect) | Effort truth, injury-risk signal | Wearable owners are the highest-LTV cohort; gate *analysis*, never raw recording |
| **Training load & recovery guidance** (weekly load ratio, rest-day nudges) | Don't get injured — the #1 reason runners quit (and churn) | Retention insurance; a WHOOP-style "readiness" number is a daily-open habit loop |
| **Route library + elevation profiles** (save/share club routes, grade-adjusted pace) | Compare like-for-like efforts; discover club routes | Shared routes are a viral surface; grade-adjusted pace is a plus analytic |
| **Shoe mileage tracker** (log pairs, wear-out alerts at ~600 km) | Replace shoes before injury | The cleanest affiliate surface in running — a wear-out alert with a recommendation converts honestly |

## Tier 2 — Social/competitive tracking that compounds retention (M4)

| Feature | Runner value | Business value |
|---|---|---|
| **Club challenges** (monthly km ladders, attendance streaks, team vs team) | Crew accountability, friendly stakes | Club Pro feature for organizers; **sponsored challenges** are the first brand-revenue surface |
| **Segments on club routes** (fastest km on the Tuesday loop) | Local bragging rights | Classic engagement flywheel; city/country leaderboards later feed premium discovery |
| **Attendance + consistency stats** (runs-with-crew count, show-up rate) | "You showed up 9 Tuesdays straight" | Uniquely ours — Strava can't see attendance; anchors Club Pro analytics |
| **Kudos + run photos on the timeline** | The social proof loop after every run | Feed density → session frequency; photo moments are organic acquisition on Instagram |
| **Race goal countdown** (target race, readiness vs plan) | Focus for the 16-week block | Bridges to paid **training plans + coach marketplace** (take-rate) |

## Tier 3 — Later, bigger surfaces (M6+)

Live safety location sharing (family "beacon" — premium), AI coach with voice cues during recorded runs (flagship ClubRuns+ v2), virtual races + event ticketing (take-rate), corporate wellness leagues (B2B subscriptions, highest ARPU), charity fundraising runs (brand halo, partner revenue).

## Monetization map (which revenue line each tier feeds)

- **ClubRuns+ (~A$9/mo)**: Tier 1 analytics (zones, curves, load/recovery, grade-adjusted pace) + later AI coach, safety beacon
- **Club Pro (~A$29/mo/club)**: challenges, attendance analytics, segments management, route library admin
- **Take-rate**: training plans, coach marketplace, event ticketing, virtual races
- **Affiliate/brand**: shoe tracker recommendations, sponsored challenges, corporate leagues
- Sequencing rule stands (`docs/MONETIZATION.md`): instrument first, gate only after W4 retention proves the loop. Recording and basic stats stay free forever — the network is the moat, analytics are the product.

## Explicitly rejected

Ads inside the run/record experience (poisons the premium register), selling location data (privacy by design, non-negotiable), NFTs (per AGENTS.md), paywalling raw tracking (kills the acquisition wedge).
